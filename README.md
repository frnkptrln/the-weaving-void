# The Weaving Void: A Collection of Esoteric Languages

Welcome to **The Weaving Void**, a repository dedicated to the exploration of unique computational patterns and philosophical concepts through esoteric programming languages (Esolangs).

Every language in this collection is theoretically Turing Complete and serves as a medium for unconventional problem-solving and digital artistry.

## Included Languages

| Language | Concept | Status |
| :--- | :--- | :--- |
| **[Distaff](./Distaff)** | Musical weaving via 4-note sequences. | Stable |
| **[Vortex](./Vortex)** | High-speed Brainfuck successor with memory warping. | Implemented |
| **[MandelMemory](./MandelMemory)** | Chaos-driven navigation on the Mandelbrot set. | Implemented |
| **[SFract](./SFract)** | Biological "seed" growth via L-Systems. | Implemented |
| **[Entanglement](./Entanglement)** | Quantum-inspired linked logic pairs. | Implemented |

## Core Implementation Principles

1.  **Turing Completeness:** All languages are designed to simulate a Turing machine's capability.
2.  **Robust Interpreters:** Written in Python 3.10+, featuring clear error handling.
3.  **Comprehensive Docs:** Each language has its own README with syntax and memory models.
4.  **Verification:** Every language includes a "Hello World" or proof-of-concept example.

## Getting Started

To explore a language, navigate to its directory and follow the instructions in the local `README.md`.

Generally, you can run an interpreter using:
```bash
python3 <LanguageFolder>/interpreter.py <SourceFile>
```
